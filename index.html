How to Apply CSS to HTML

CSS stands for “Cascading Style Sheets” and it is used to specify the style and presentation of HTML elements. CSS is written out on a separate file and connected to an HTML file using the < link > tag in the < head > section of the HTML file. The < link > tag provides the path to the CSS file and it also includes “rel” and “type”. The relationship between the CSS file and the HTML file is put in the “rel” area and “type” describes the format of the CSS file. The example below shows the < link > tag used for this specific HTML page and the CSS code in the seperate file used to alter the margin, padding, and line-height of the paragraph.


< link href="file:///Users/cyrusperez/Desktop/assignment-2-repo-1/resources/layout.css" rel="stylesheet" type="text/css">
.css-question { margin: 5px; padding: 5px; line-height: 2; }

In this code, "margin" has created a space outside of the paragraph element and other elements. The "padding" element has created a space in between the border of the paragraphs contents its border. As for the "line-height", this feature adjusts the space between lines in the paragraph.
