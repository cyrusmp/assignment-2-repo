How to Apply CSS to HTML

CSS stands for "Cascading Style Sheets" and it is added to HTML files to help style the webpage. To include CSS elements in an HTML file, you would first create a CSS file. The file would include a reference to a class name with brackets that will then enclose the attributes one wishes to add. The following is an example of CSS code that was used for this paragraph.


.css-question { margin: 0; padding: 5px; line-height: 2; }

In this code, "margin" has created a space between the paragraph and the screen. The "padding" element increases or decreases the space in between the paragraph and the heading. As for the "line-height", this feature adjusts the space between lines in a paragraph. The CSS information was added to the HTML file referencing this webpage using a link tag. The link tag includes the path to the CSS file along with the "rel" and "type".
